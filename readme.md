# NFC Redirect Server (Raspberry Pi)

A lightweight LAN-based redirect controller designed for churches using NFC tags.  
Allows instantly changing the redirect URL from a clean web interface.

## Features
- Redirect endpoint for NFC tags
- Simple HTML/CSS UI
- Preset URLs with one-click switching
- Temporary URL overrides with expiration
- Runs as a background service on Raspberry Pi
- Zero framework front-end
